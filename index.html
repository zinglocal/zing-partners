ZING Partner Manager
One-time setup — ~10 minutes
Step 1 — Create your Google Sheet
Go to sheets.google.com, create a blank sheet called ZING Partner Manager. Copy the Sheet ID from the URL (long string between /d/ and /edit).
Step 2 — Add the Apps Script
In your Sheet: Extensions → Apps Script. Paste the code from apps-script.gs. Click Deploy → New deployment → Web app, set access to Anyone. Copy the URL.
Step 3 — Connect Stripe
In Stripe Dashboard → API keys, copy your Publishable key. Your Secret key goes in the Apps Script only.
